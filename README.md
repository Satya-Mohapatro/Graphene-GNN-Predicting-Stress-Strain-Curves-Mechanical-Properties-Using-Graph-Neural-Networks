# Graphene-GNN-Predicting-Stress-Strain-Curves-Mechanical-Properties-Using-Graph-Neural-Networks
End-to-end GNN pipeline for predicting mechanical behavior of defected graphene. Includes synthetic data generation, graph construction, property prediction (Young’s modulus &amp; fracture strain), and full 801-point stress–strain curve prediction with R² ≈ 0.9925.
